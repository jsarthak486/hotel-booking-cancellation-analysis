# Hotel Booking Cancellation Analysis (Excel Project)

## Project Overview
This project analyzes hotel booking cancellation patterns using Microsoft Excel. The objective is to identify the major factors affecting booking cancellations and provide useful business insights to improve hotel reservation management.

---

## Objective
The main goals of this project are:

- Analyze hotel booking cancellation trends
- Identify factors affecting cancellations
- Compare cancellation behavior across hotel types
- Understand customer booking patterns
- Create an interactive Excel dashboard

---

## Dataset Information
The dataset contains hotel booking information including:

- Booking Status
- Hotel Type
- Customer Type
- Lead Time
- Stay Duration
- Country
- Deposit Type
- Arrival Date
- Number of Guests

---

## Dashboard Insights
Some key findings from the analysis:

- City hotels showed higher cancellation rates compared to resort hotels.
- Customers with higher lead time were more likely to cancel bookings.
- Non-refundable deposits reduced cancellation probability.
- Seasonal trends affected booking behavior.
